# figwatch-olivia

Deployment stack based on [simonpforster/figwatch](https://github.com/simonpforster/figwatch).

Includes figwatch, Cloudflare Tunnel, OpenTelemetry Collector, Prometheus, and Grafana.

For figwatch configuration details, see [simonpforster/figwatch](https://github.com/simonpforster/figwatch).
